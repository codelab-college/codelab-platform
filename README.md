# CodeLab Platform

An internal academic coding platform for college students to solve programming assignments and participate in contests.

## Tech Stack

### Frontend
- React (functional components only)
- Tailwind CSS v3.x
- Mobile-first design
- No external component libraries

### Backend
- Node.js with Express.js
- SQLite3 database
- REST APIs
- Role-based access control (Student / Teacher / Admin)
